# Char2D

A game engine and game demo built in beeflang.

## Documentation

The website is built using [Docusaurus](https://docusaurus.io/) and is isolated within the `docs/` folder.

### Installation and Local Development

To view and edit the documentation locally on your machine:

1. Navigate to the documentation folder:
```bash
cd docs
```

2. Install the dependencies (only needed the first time):
```bash
npm install
```

3. Start the development server:
```bash
npm start
```
This command starts a local development server and you can acess it by going to [This Address](http://localhost:3000/Char2D/). Any changes made to the Markdown files will be reflected live without having to restart the server.
