# Char2D

A game engine and game demo built in beeflang.

## Documentation & How to Run the Project Locally

The website is built using [Docusaurus](https://docusaurus.io/) and is isolated within the `docs/` folder.

For complete contribution guidelines, coding standards, and detailed instructions on how to run both the engine and the documentation site locally, please read the [How to Contribute & Run](docs/docs/contributing.md) guide.
