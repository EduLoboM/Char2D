# Welcome to the Char2D Project

Welcome to the official documentation of the **Char2D** project. This space is dedicated to centralizing all information, artifacts, and architectural decisions about our custom Game Engine and the Game itself.

## Document Structure

This documentation is divided into three main pillars:

1. **[Game Engine](engine/index.md):** Technical documentation focused on the engine that powers the game. Includes details about architecture, systems (rendering, physics, input), and API reference.
2. **[The Game](game/index.md):** Information about the game itself, including the Game Design Document (GDD), mechanics, story, characters, etc.
3. **[Artifacts and Assets](artifacts/index.md):** Organization of resources, visual assets, sound assets, scripts, and other important artifacts for development.
